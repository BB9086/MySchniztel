# 🍔 MySchnitzel Food Ordering App

A mobile food ordering application that allows users to order meals directly from the nearest restaurant location.

After logging in, users can select a nearby store, browse the menu, add products to the shopping basket, and complete their order through a simple and user-friendly interface.

The application uses location services to calculate the distance between the user and available stores. Push notifications are used to inform users when their order has been processed.

---

# ✨ Features

- User registration and authentication
- Password recovery functionality
- Store locator with distance calculation
- Food menu browsing
- Food category filtering
- Shopping basket management
- Push notifications for order updates
- Responsive mobile user interface

---

# 🛠️ Technologies Used

- Xamarin Native
- ASP.NET Web API
- MS SQL Server
- Push Notifications
- Geolocation Services
